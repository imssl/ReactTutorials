UI TEMPLATES

GOAL

Create a minimized version of a free Css+Html template.
Minimized version should have following features:

  • All files in one folder
    I can zip/unzip this folder and copy it everywhere

  • Should contain minimal example pages:
    – Main layout
    – Login
    – Page with a form
    – Showcase of UI html components (at least text, checkboxes)

  • Shouldn’t require any build operation from tools in order to work:
    –Means no npm install or gulp or any build tool needed
    –All (minimal javascript,css) should be linked in pages

  • Folder should be under 10Mb, (best would be under 5Mb)

How to proceed…?

  • Keep one copy of the original untouched (best would be to use git) -
  • Keep few pages (more pages, more dependencies)
  • For each page try to comment out links one by one:
    – Media files (img, video, animations)
    – Js to external libraries (UI components, example code)
    – Css, Scss, sass
  • Check if elements in page are still displayed correctly
    Interaction and charts will probably disappear or break, but this is not a problem at the moment, you’ll replace it later with yours.
  • If it works in one page, most likely links to remove are the same in all the other you selected.
  • Copy pages selected, and the few resources you decided to keep in one new folder, and try it out. This is the target folder, what should be under 10Mb.
